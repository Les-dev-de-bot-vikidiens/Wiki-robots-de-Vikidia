The **Multics Relational Data Store**, or **MRDS**, was the first commercial [relational database management system](/broken/pages/cbebe8ef792bb0ddb42e39b56eb4353fe6845c38). It was written in [PL/1](/broken/pages/4c27f6f80d8d7bfaabb321569f49f3b8e7290590) by [Honeywell](/broken/pages/74e1970054a9028a78db090486ee35d3d1a084d7) for the [Multics](/broken/pages/0034633dc4920f8ee911ee0a60f487f611585ea1) [operating system](/broken/pages/bbb3cf2d62b6eadb5d7bc55b8038c29cd689f14f) and first sold in June 1976.\[1] Unlike the [SQL](/broken/pages/0b1fff78eb8d441356b51f0652806d0a06e8f485) systems that emerged in the late 1970s and early 80s, MRDS used a command language only for basic data manipulation (equivalent to the `SELECT` or `UPDATE` statements in SQL). Other operations, such as creating a new database or general file management, required the use of a separate command program.
